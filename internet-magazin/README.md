# Интернет-магазин на Python

Простой интернет-магазин на Flask с базой данных SQLite, каталогом товаров, корзиной и оформлением заказа.

## Установка

1. Установите зависимости:

```bash
pip install -r requirements.txt
```

2. Запустите приложение:

```bash
python app.py
```

3. Откройте в браузере:

```text
http://127.0.0.1:5000
```

## Возможности

- Просмотр списка товаров
- Детали товара
- Добавление товара в корзину
- Удаление товара из корзины
- Регистрация и вход пользователей
- Оформление заказа
- История заказов в профиле
- **Админ-панель для управления товарами** (добавление, редактирование, удаление)
- **Управление пользователями** для роли admin
- Смена пароля для каждого пользователя
- Постоянное хранение данных в SQLite базе данных

## Администратор

Админский доступ ограничен пользователем `admin`.

По умолчанию создается аккаунт:
- Логин: `admin`
- Пароль: `admin123`

Можно изменить эти данные в `app.py` в переменных `ADMIN_USERNAME` и `ADMIN_PASSWORD`.

## База данных

Приложение использует SQLite базу данных (`store.db`) для хранения:
- Товаров
- Пользователей
- Заказов

База данных создается автоматически при первом запуске.
